# 学习JAVA

这是我的Java学习代码仓库，包含了学习Java过程中的各种练习和示例代码。

## 目录结构

- `src/` - 源代码目录
  - `algorithm/` - 算法相关代码
  - `cms/` - CMS系统相关代码
  - `day*_teacher_code/` - 教学代码
  - `day_*/` - 按天分类的练习代码，包含day_08到day_15等目录
- `images/` - 学习过程中的示例图片
- `lib/` - 第三方库（如lombok）

## 学习内容

- Java基础语法（变量、数据类型、运算符、流程控制等）
- 面向对象编程（类与对象、继承、多态、封装、抽象类、接口）
- 数据结构与算法基础
- 异常处理机制
- 内部类与枚举
- Arrays类与集合框架（List、Set、Map等）
- 泛型编程
- 日期时间处理
- StringBuffer与StringBuilder
- StringJoiner工具类
- 数学工具类

## 项目特点

- 按天分类，便于学习和复习
- 包含各种练习和示例代码
- 涵盖Java核心知识点
- 持续更新中...
